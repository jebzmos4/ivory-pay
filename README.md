Ivory Pay Assessment
==============================================

This Repository contains the Restaurant API and Algorithm Task.
----------

Clone repository and run npm install to setup dependencies

No need for a `.env` file
----------------------------

Build Solution
----------------------------
```
npm run build
```
Use this in development mode
-------------
```
npm run dev
```
Get API running
-------------
```
npm run start
```
